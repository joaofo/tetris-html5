# Tetris — Lean Six Sigma Edition (HTML5)

A lightweight Tetris clone built with Canvas.

## Theme
Progress through Lean Six Sigma concepts:
- Level 1–3: Green Belt
- Level 4+: Black Belt (Manufacturing)

Concepts are written on the bricks as you build your stack.

## Controls
- Left/Right: move
- Up: rotate
- Down: soft drop
- Space: hard drop
- P: pause
